# Property Siege Classic

This is the first playable scenario pack for Chatty Quest.

`v0.1` treats the player's property as the full playable map.

Important:

- leaving the property is a scenario rule, not a universal engine law
- the narrator may express the boundary with tone and humor
- the reducer and scenario rules still own the truth

This pack is currently scaffolded with placeholder template files and capsule text. Real content loading comes in the datapack implementation phase.
